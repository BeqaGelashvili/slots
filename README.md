# Neon Slot Machine 🎰

A vibrant, interactive slot machine game built with pure JavaScript. Experience a retro-casino vibe right in your browser!

## 🚀 Features
* **Custom Game Logic:** Real-time payout calculations and reel randomization.
* **Sleek UI:** Neon glow effects and smooth CSS3 animations.
* **Fully Responsive:** Play on desktop, tablet, or mobile.
* **Interactive Experience:** Sound effects and dynamic balance updates.
